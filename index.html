Version française

This is the starter theme for projects of JLM 2017, the campaign of Jean-Luc Mélenchon for the French presidential election.

This theme is based on Bootstrap version 3. All default Bootstrap components (grid, buttons, etc.) are available, plus a certain number of custom components.

NPM package

npm i @fi/theme

Custom CSS components

Responsive main column

Class col-responsive is a short-hand for col-sm-12 col-md-10 col-md-offset-1 col-lg-8 col-lg-offset-2. The central column on this page use col-reponsive.

Custom JS components

Responsive videos

Include any Youtube or Facebook video with an iframe. It will resize to the parent container width.
